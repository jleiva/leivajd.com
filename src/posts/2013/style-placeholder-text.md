Snippet recomendado, [darle estilo a placeholder](http://css-tricks.com/snippets/css/style-placeholder-text/).

```css
::-webkit-input-placeholder {
  color: #333;
}

:-moz-placeholder {
  /* Firefox 18- */
  color: #333;
}

::-moz-placeholder {
  /* Firefox 19+ */
  color: #333;
}

:-ms-input-placeholder {
  color: #333;
}
```
